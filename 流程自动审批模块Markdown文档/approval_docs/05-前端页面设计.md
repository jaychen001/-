# 前端页面设计

## 1. 页面结构总览

第一阶段需要开发以下页面：

```text
审批中心
├─ 我的待审批
├─ 我发起的审批
├─ 抄送我的审批
├─ 审批详情页
│
ECN 变更审批
├─ ECN 列表页
├─ 新建 ECN
├─ 编辑 ECN
├─ ECN 详情页
│
新项目下单审批
├─ 新项目下单列表页
├─ 新建项目下单申请
├─ 编辑项目下单申请
├─ 新项目下单详情页
│
系统配置
├─ 审批模板配置
├─ 审批节点配置
```

## 2. 审批中心

### 2.1 我的待审批

功能：

- 展示当前登录用户需要审批的任务。
- 支持按审批类型筛选。
- 支持按提交时间排序。
- 点击记录进入审批详情页。

表格字段：

| 字段 | 说明 |
|---|---|
| 审批标题 | 例如 ECN变更审批-ECN20260001 |
| 审批类型 | ECN / 新项目下单 |
| 申请人 | 发起人 |
| 当前节点 | 当前待审批节点 |
| 提交时间 | 申请提交时间 |
| 状态 | 审批中 |
| 操作 | 查看 / 审批 |

筛选条件：

- 审批类型
- 申请人
- 提交时间
- 关键字

### 2.2 我发起的审批

功能：

- 查看自己提交的所有审批。
- 支持按状态筛选。
- 支持进入详情查看审批进度。

表格字段：

| 字段 | 说明 |
|---|---|
| 审批标题 | 审批名称 |
| 审批类型 | ECN / 新项目下单 |
| 当前状态 | 草稿/审批中/已通过/已驳回/退回修改 |
| 当前节点 | 当前审批节点 |
| 提交时间 | 提交时间 |
| 完成时间 | 审批完成时间 |
| 操作 | 查看 / 编辑 / 撤回 |

操作规则：

- 草稿状态可以编辑。
- 退回修改状态可以编辑。
- 审批中状态可以查看和撤回。
- 已通过、已驳回状态只能查看。

### 2.3 抄送我的审批

功能：

- 查看抄送给当前用户的审批。
- 支持已读/未读状态。
- 点击后进入详情页并自动标记已读。

表格字段：

| 字段 | 说明 |
|---|---|
| 审批标题 | 审批名称 |
| 审批类型 | ECN / 新项目下单 |
| 申请人 | 发起人 |
| 审批结果 | 已通过/已驳回 |
| 抄送时间 | 抄送生成时间 |
| 阅读状态 | 已读/未读 |
| 操作 | 查看 |

## 3. 审批详情页

审批详情页是通用页面，ECN 和新项目下单共用。

页面区域：

```text
审批详情页
├─ 基础信息区
├─ 业务表单区
├─ 审批进度区
├─ 审批日志区
├─ 操作按钮区
```

### 3.1 基础信息区

显示：

- 审批标题
- 审批类型
- 审批状态
- 申请人
- 提交时间
- 完成时间

### 3.2 业务表单区

根据 businessType 展示不同业务内容。

如果是 ECN_APPROVAL，展示 ECN 变更字段。

如果是 PROJECT_ORDER_APPROVAL，展示新项目下单字段。

### 3.3 审批进度区

建议使用步骤条展示。

字段：

| 字段 | 说明 |
|---|---|
| 节点名称 | 项目经理审批/研发负责人审批 |
| 审批人 | 节点审批人 |
| 节点状态 | 待审批/已通过/已驳回 |
| 审批意见 | 审批人填写 |
| 审批时间 | 操作时间 |

### 3.4 审批日志区

按时间倒序展示日志。

字段：

| 字段 | 说明 |
|---|---|
| 操作时间 | 日志时间 |
| 操作人 | 谁操作 |
| 操作动作 | 提交/通过/驳回/退回/撤回 |
| 操作意见 | 审批意见 |
| 状态变化 | from_status → to_status |

### 3.5 操作按钮区

根据当前用户身份和状态展示不同按钮。

#### 当前用户是审批人，且当前节点待审批

显示：

- 通过
- 驳回
- 退回修改

#### 当前用户是申请人，且状态为草稿

显示：

- 编辑
- 提交审批

#### 当前用户是申请人，且状态为退回修改

显示：

- 编辑
- 重新提交

#### 当前用户是申请人，且状态为审批中

显示：

- 撤回

#### 其他情况

只显示：

- 返回

## 4. ECN 变更审批页面

### 4.1 ECN 列表页

表格字段：

| 字段 | 说明 |
|---|---|
| ECN 编号 | ECN 单号 |
| 项目名称 | 所属项目 |
| 产品名称 | 产品/设备 |
| 变更类型 | 图纸/BOM/物料/工艺 |
| 状态 | 当前状态 |
| 申请人 | 发起人 |
| 创建时间 | 创建时间 |
| 操作 | 查看 / 编辑 / 提交 |

按钮：

- 新建 ECN
- 查询
- 重置

筛选：

- ECN 编号
- 项目名称
- 变更类型
- 状态
- 申请人

### 4.2 新建/编辑 ECN 页面

表单字段：

- 项目名称
- 产品名称
- 变更类型
- 变更原因
- 变更内容
- 影响范围
- 涉及图纸
- 涉及 BOM
- 涉及物料
- 是否影响成本
- 是否影响交期
- 是否影响库存
- 附件上传

页面按钮：

- 保存草稿
- 提交审批
- 返回

校验要求：

1. 项目名称必填。
2. 变更类型必填。
3. 变更原因必填。
4. 变更内容必填。
5. 影响范围必填。
6. 成本、交期、库存影响必须选择。

## 6. 审批配置页面

第一阶段可以做简单配置页，也可以先使用初始化数据。

### 6.1 审批模板列表

字段：

| 字段 | 说明 |
|---|---|
| 模板名称 | ECN 变更审批 |
| 模板编码 | ECN_APPROVAL |
| 业务类型 | ECN_APPROVAL |
| 是否启用 | 是/否 |
| 操作 | 编辑 / 启用 / 停用 |

### 6.2 审批节点配置

字段：

| 字段 | 说明 |
|---|---|
| 节点顺序 | sort_order |
| 节点名称 | 审批节点 |
| 审批角色 | 项目经理/研发负责人/总经理 |
| 是否必审 | 是/否 |
| 操作 | 编辑 / 删除 |

## 7. 页面权限

1. 普通用户只能查看与自己有关的审批。
2. 审批人只能审批待自己审批的节点。
3. 抄送人只能查看，不能审批。
4. 管理员可以查看全部审批。
5. 管理员可以进入审批配置页面。
